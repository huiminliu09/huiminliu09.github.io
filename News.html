E- News



This is a simple news search engine, similar to Google News. Users can search for some News that we obtained from the Common Crawl in the search bar. Then our search engine will return results.

Learning objectives:

  • Explore the AWS management console
  • Read the documentation for a REST api and write code that makes REST requests
  • Get comfortable developing Java applications using some professional software engineering tools, such as the Maven build system

Description:

This project mainly includes three parts:

  • News Article ETL: download a WARC from Common Crawl, and loaded them into Elastic Search.
  • Search Backend Service: provide a REST API to search the news article database
  • Search Frontend: a basic React Javascript frontend to present search results
